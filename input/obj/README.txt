# 001_classroom - Geometric Model

This folder contains the geometric and reference documentation for the classroom model.

Contents:
- classroom1.obj / .mtl / .skp → 3D geometry for Treble import
- classroom1.pdf → dimensioned drawing for reference
- geo_notes.txt → notes on coordinate system, scaling, and material tagging

Purpose:
Provides the forward model geometry for Treble hybrid simulations.
